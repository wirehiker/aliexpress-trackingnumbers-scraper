# aliexpress-trackingnumbers-scraper 
Javascript script for the browser console that lists the tracking numbers of the items on the oders list. 

## How to use: 
* go to orders 
* select the list of orders from which you want to retrieve the tracking numbers 
* open the developer tools of the browser 
* open the console 
* paste the script and run it 

# the script requires the right to open pop-up windows 

The found tracking numbers are lists one the script is done.
